# Vercube Nitro Example

A example showcasing how to use Vercube with Nitro. This template demonstrates how to build applications using Nitro as a server framework, making it easy to leverage Nitro's file-based routing and server capabilities alongside Vercube.

## <a name="getting-started">🚀 Quick Start</a>

```sh
$ pnpm i
$ pnpm dev
```

Visit http://localhost:3000/api in your browser, and you should see your Vercube application running!
