Zero configuration to get started

Vercube works right out of the box with sensible defaults that help you create production-ready applications quickly. No complex setup required just install and start building.

When you need more control, every aspect of Vercube can be customized through a simple, intuitive configuration API that keeps your codebase clean and maintainable.

Compose with everything you need.

Auth

Complete authentication system with support for JWT, sessions, OAuth, and role-based access control to secure your applications.

CLI

Powerful command-line tools to scaffold projects, generate components, and manage your application with intuitive commands.

Core

Essential utilities and base components that form the foundation of Vercube, providing optimized performance and developer experience.

Logger

Advanced logging system with customizable log levels, formatters, and transports to help debug and monitor your application in any environment.

Storage

Universal storage API with providers for Redis, Cloudflare KV, Amazon S3, in-memory storage and more, all with a consistent interface.

Runtime Agnostic

Deploy anywhere with runtime-agnostic architecture supporting Node.js, Deno, Bun, serverless platforms, and edge computing environments.

IOC

Powerful dependency injection system that simplifies testing, promotes modular design, and enables clean separation of concerns in your application.
